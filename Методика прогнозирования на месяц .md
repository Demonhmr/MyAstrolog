
# Методика прогнозирования на месяц (лунар)

Для проведения полноценного расчета по методике прогнозирования на месяц (лунар) вам потребуются определенные входные данные и последовательное выполнение математических и аналитических шагов.

## Данные, необходимые для расчета

Для работы методики требуется подготовить следующую базу данных и индивидуальные показатели:

### Индивидуальные астрологические данные

- **Натальная карта человека** (внутренний круг для расчетов)
- **Лунар (лунное возвращение)**: момент, когда транзитная Луна возвращается в те же координаты (знак, градус, минуты, секунды), в которых она была при рождении


### Справочные таблицы (база данных)

- **Свойства знаков зодиака**: к какой стихии (Огонь, Земля, Воздух, Вода) и какому кресту (Кардинальный, Фиксированный, Мутабельный) относится каждый из 12 знаков
- **Свойства домов натальной карты**: к какой стихии и кресту привязан каждый дом (сфера жизни)
- **Веса (баллы) планет**: каждая планета имеет свой «вес»
    - Солнце и Луна — по **5 баллов**
    - Меркурий, Венера и Марс — по **3 балла**
    - Юпитер и Сатурн — по **2 балла**
    - Уран, Нептун и Плутон — по **1 баллу**
    - *Общая сумма баллов для проверки должна составлять **26***
- **Текстовые интерпретации**: ключевые фразы для каждой стихии, креста, знака зодиака и дома


## Пошаговая инструкция по использованию методики

### Шаг 1. Построение карты лунара

Необходимо построить карту на текущий месяц, зафиксировав положение планет в момент возвращения Луны в натальную позицию.

### Шаг 2. Расчет баллов по знакам (внутреннее состояние)

Этот этап определяет настроение, потребности и ожидания человека на месяц.

1. Выпишите все планеты и знаки, в которых они находятся в лунаре
2. Присвойте каждой планете балл согласно ее весу
3. Суммируйте баллы отдельно для каждой стихии и каждого креста, основываясь на свойствах знаков, в которых стоят планеты
4. **Проверка**: Сумма всех баллов по знакам должна быть равна **26**

### Шаг 3. Расчет баллов по домам (внешние обстоятельства)

Этот этап определяет события, задачи и обстоятельства, с которыми придется столкнуться.

1. Определите, в какие дома лунара попали планеты
2. Суммируйте баллы планет для каждой стихии и креста, основываясь на свойствах этих домов
3. **Проверка**: Сумма баллов по домам также должна составить **26**

### Шаг 4. Определение доминирующих факторов и синтетических показателей

1. Выявите стихию и крест, набравшие **максимальный балл** в расчете по знакам:
    - **Стихия по знаку** — это **темперамент** человека на данный месяц
    - **Крест по знаку** — это **предпочитаемый способ действий** человека
2. Выявите стихию и крест с **максимальным баллом** в расчете по домам:
    - **Стихия по дому** — это **характер событий**, с которыми столкнется человек
    - **Крест по дому** — это **динамика событий** месяца
3. Вычислите **синтетический знак**: найдите знак, объединяющий доминирующую стихию и крест из расчета по знакам — это **роль-архетип поведения** на месяц
4. Вычислите **синтетический дом**: найдите дом, соответствующий доминирующей стихии и кресту из расчета по домам — это **главная цель и задача месяца**

### Шаг 5. Интерпретация и анализ конфликтов

1. **Сравните результаты**: если доминирующие стихии по знакам и домам различаются, это указывает на **внутренний конфликт** (например, человек хочет быть мягким по стихии Воздуха, а обстоятельства требуют жесткости по Огню)
2. Используйте ключевые фразы для описания:
    - «**Как я хочу**» (знаки)
    - «**Как будет на самом деле**» (дома)
3. Определите **динамику месяца**:
    - **Асцендент (Asc)** лунара — характеризует **начало месяца**: знак Asc описывает настроение и стиль поведения, а дом, в который попал **управитель Asc**, указывает на обстоятельства и сферу жизни, с которыми это связано
    - **MC (середина неба)** — характеризует **завершение месяца**: знак MC описывает итоговое состояние и стиль, а дом, в который попал **управитель MC**, указывает на обстоятельства и сферу жизни, с которыми связан итог

## Возможности методики

Методика позволяет не только давать **общий прогноз**, но и **адаптировать его под конкретный запрос пользователя** (работа, деньги, отношения), используя полученные данные о доминирующих стихиях и крестах.
